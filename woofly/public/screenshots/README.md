# Screenshots pour PWA

Pour une meilleure présentation dans les stores d'applications et lors de l'installation,
ajoutez des captures d'écran de votre application ici.

## Tailles recommandées:

### Mobile (narrow)
- 540x720px (portrait)
- Capturez les écrans principaux de votre app

### Desktop (wide)
- 1280x720px (paysage)
- Vue d'ensemble de l'interface desktop

## Comment créer les screenshots:

1. Ouvrez votre application dans un navigateur
2. Utilisez les outils de développement (F12)
3. Activez le mode responsive
4. Prenez des captures des écrans clés:
   - Page d'accueil
   - Profil du chien
   - Forum/Social feed
   - Page adoption

Nommez-les:
- `mobile-1.png`, `mobile-2.png`, etc.
- `desktop-1.png`, `desktop-2.png`, etc.
